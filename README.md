# Library Management System In Java

Make an application for a library. 
- It should be possible to add books to the library with author name and title as well as other useful information. It should be possible to remove books as well. 
- It should be possible to have multiple copies of books in the library. 
- Should be able to register user “Multiple” to library
- User should be able to borrow book
- User should be able to return book 
- Library should be able to keep track of people who borrowed books and how many books they have borrowed and which book they have borrowed.
- Users should be able to return book to the library.
- User should NOT be able to borrow more than 5 books at a time;
- User should NOT be able to borrow more than one copy of the same book
- It should be possible to see statistic of all books
- It should be possible to search for books by author or name
- It should be possible to see all books that are more than 0 copies left in library when user wants to borrow
